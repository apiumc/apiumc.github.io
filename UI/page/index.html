与第三方应用身份对接是采用网关技术实现的解决方案，兼容企业现有应用，各身份对接配置是非侵入在线配置即可，应用无感知，成本减化看的见，企业内部协调、推广更容易。

它不破坏现有应用账号管理体系

零信任网关与应用采用的是身份转化合成带身份请求，不强制应用账号与网关身份的唯一性，因此支持应用账户多人使用，但都会被记录，使现有成熟账号管理经验被保留，更符合企业真实业务场景。

丰富的第三方登录，身份认证更加简单

预集成多种外部认证源，简单配置开箱即用, 支持AD、LDAP等本地认证登录，也支持外部认证源扫码登录，而各应用不做改动就可集成，从而让企业各应用登录入口从分散各处到统一扫码登录，简化员工的工作。
